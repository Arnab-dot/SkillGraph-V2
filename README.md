# SkillGraph

SkillGraph is an end-to-end intelligence platform that extracts, analyzes, clusters, and forecasts skills from job descriptions and resumes. It uses advanced Machine Learning to build a co-occurrence graph of skills, identify market trends, and provide gap analysis for job seekers.

## Features

- **Skill Extraction & Parsing**: Extracts skills from job postings and PDF resumes.
- **Topic Modeling & Clustering**: Groups related jobs and skills using BERTopic, UMAP, and HDBSCAN.
- **Skill Co-occurrence Graph**: Builds relationships between skills to show how they are commonly paired.
- **Demand Forecasting**: Predicts future skill demand using a TensorFlow LSTM model.
- **Anomaly Detection**: Flags unusual job postings using a PyTorch Autoencoder.
- **Resume Analyzer**: Compares an uploaded resume against target roles (like ML Engineer or Data Scientist) to provide a gap analysis of matching vs. missing skills.

## Architecture

The project is split into three main components:

### 1. Machine Learning Pipeline (`/ml`)
The ML pipeline processes raw job data to generate insights, models, and JSON outputs.
- **Stack**: Python, PyTorch, TensorFlow, BERTopic, UMAP, HDBSCAN, Scikit-learn.
- **Outputs**: Generates graph definitions, trend data, clusters, and forecasting reports saved to `ml/outputs/`.

### 2. Backend (`/backend`)
A RESTful API that serves the data generated by the ML pipeline to the frontend and handles live resume parsing.
- **Stack**: FastAPI, SQLAlchemy, SQLite, Uvicorn.
- **Key Endpoints**:
  - `/api/dashboard`: Summary statistics.
  - `/api/graph`: Skill co-occurrence network data.
  - `/api/trends`, `/api/forecast`: Time-series data for skill demand.
  - `/api/resume/analyze`: Live PDF parsing and gap analysis.

### 3. Frontend (`/frontend`)
A modern, interactive dashboard for exploring the skill graph, clusters, forecasts, and analyzing resumes.
- **Stack**: Vite, React, TypeScript, Recharts, React Flow (for graph visualization).

## Setup & Execution Instructions

### Prerequisites
- Python 3.10+
- Node.js 18+

### 1. Environment Setup (ML & Backend)
We recommend using a Python virtual environment to manage dependencies.

```bash
# Create and activate virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r backend/requirements.txt
pip install -r ml/requirements.txt
```

### 2. Running the ML Pipeline
Execute the pipeline to process sample job data and generate all required models and reports.

```bash
# Ensure your virtual environment is active
source .venv/bin/activate

# Run the pipeline
python3 ml/run_pipeline.py --input ml/data/sample/sample_jobs.csv
```

### 3. Starting the Backend Server
Start the FastAPI server. It reads the pre-computed outputs from the ML pipeline.

```bash
# Ensure your virtual environment is active
source .venv/bin/activate

# Start the server (runs on http://localhost:8000)
uvicorn backend.app.main:app --host 0.0.0.0 --port 8000
```

### 4. Starting the Frontend
Install Node dependencies and start the Vite development server.

```bash
cd frontend
npm install
npm run dev
```

Visit the local URL provided by Vite (usually `http://localhost:5173`) to view the dashboard.
